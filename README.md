# dorm-laundry

寮の共同洗濯機向けフロントエンドです。`AGENTS.md` に沿って、Next.js App Router / TypeScript / Tailwind CSS / TanStack Query を前提に最小構成を作成しています。

## 画面

- `/` 洗濯機一覧
- `/machines/[machineId]` 洗濯機詳細と利用開始導線
- `/sessions/current` 自分の利用中セッション
- `/settings/notifications` 通知設定
- `/admin` 最小管理画面

## 開発

```bash
npm install
npm run dev
```

## テスト

```bash
npm test
```
